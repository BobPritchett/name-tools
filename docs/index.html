name-tools

A utility library for parsing, formatting, and manipulating person names

Interactive Demo

The parser returns the original string fields (prefix, first, middle, last, suffix, nickname) plus additive structured metadata: prefixTokens, suffixTokens, familyParticle, familyParts, familyParticleBehavior, preferredGiven, and sort.

Installation

pnpm add name-tools

Quick Start

import { parseName, formatName } from 'name-tools';

const parsed = parseName("Dr. Bob Pritchett Jr.");
console.log(parsed);
// { prefix: 'Dr.', first: 'Bob', last: 'Pritchett', suffix: 'Jr.' }

console.log(formatName("Dr. Bob Pritchett Jr."));
// "Bob Pritchett, Jr." (NBSP used for nicer line breaks)

console.log(formatName("Dr. Bob Pritchett Jr.", { preset: "alphabetical" }));
// "Pritchett, Bob, Jr."

Features

  • Parse Names: Extract components (prefix, first, middle, last, suffix)
  • Format Names: Multiple formatting options (Last-First, First-Last, with initials)
  • Extract Data: Get first name, last name, initials
  • Data Sets: Includes common prefixes (Mr., Dr., etc.) and suffixes (Jr., PhD, etc.)
  • TypeScript Support: Full type definitions included
  • Zero Dependencies: Lightweight and fast

API Reference

Parsing Functions

  • parseName(fullName: string) - Parse a full name into components
  • getFirstName(fullName: string) - Extract first name
  • getLastName(fullName: string) - Extract last name

Formatting Functions

  • formatName(input, options?) - Single rendering entry point (presets + options)

Data & Utilities

  • PREFIXES - Array of common name prefixes
  • SUFFIXES - Array of common name suffixes
  • isPrefix(str: string) - Check if string is a prefix
  • isSuffix(str: string) - Check if string is a suffix
